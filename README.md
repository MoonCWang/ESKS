# Kitty Friends App
Demo the app here: http://mernmaven.com/portfo/kittyfriends/
![Screen of Kitty Friends App](/screen.jpg?raw=true)

## To run this project
1) Clone this repo
2) Edit package.json's homepage if you are not building
`"homepage": "http://yourwebsite.com/kittyfriends", `
3) Run npm install
4) Run npm start
